# Taxi24-API


## Description

This application showcases an example solution for managing and assigning taxi rides to drivers based on passenger requests, utilizing proximity-based search between the passenger and the taxi.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
